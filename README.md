
# 4-Wheeler Booking Frontend

## Overview
A frontend application for booking 4-wheelers with location-based services and vehicle selection.

## Features

- **Location Selection**: Pick-up and drop-off locations
- **Distance Calculation**: Automatic distance estimation
- **Service Options**: Multiple service categories
- **Vehicle Selection**: Choose from available car options
- **Contact Form**: Get in touch with support team

## How to Use

1. Select pickup location
2. Select drop-off location
3. View calculated distance
4. Choose preferred service
5. Select vehicle type
6. Fill contact details in the form
7. Submit booking request
